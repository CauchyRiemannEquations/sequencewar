# Sequence Last Stand

수열 문제를 풀면서 병력을 모으고, 라운드별 보스를 격파하는 HTML5 캔버스 게임입니다.

## 실행 방법

`last_stand.html` 파일을 브라우저로 열면 바로 실행됩니다.

GitHub Pages에 올리는 경우에는 이 폴더의 모든 파일을 같은 위치에 업로드한 뒤, `last_stand.html`로 접속하면 됩니다.

## 조작

- 마우스 또는 터치 드래그: 사령관 이동
- 키보드 `A` / `D` 또는 `←` / `→`: 사령관 좌우 이동
- 보스전에서는 병사와 사령관이 자동으로 공격합니다.

## 게임 규칙

- 수열의 다음 값을 보고 올바른 관문을 통과합니다.
- 정답을 고르면 병력 증가, 무기 강화, 골드 보상을 얻을 수 있습니다.
- 오답을 고르면 병력을 잃거나 무기 레벨이 내려갈 수 있습니다.
- 각 라운드의 관문을 모두 통과하면 보스전이 시작됩니다.
- 총 10라운드가 있으며, 10라운드 보스가 최종 보스입니다.
- 어느 라운드에서든 실패하면 골드, 강화, 병력, 진행도가 모두 초기화되고 1라운드부터 다시 시작합니다.
